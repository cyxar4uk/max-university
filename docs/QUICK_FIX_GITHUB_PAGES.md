## Возможные проблемы:

### Проблема: "Cannot find module"
**Решение:** Все импорты должны иметь расширения `.js` или `.jsx`

### Проблема: "404 для main.jsx"
**Решение:** Vite автоматически обрабатывает пути при сборке. Убедитесь, что `base: '/max-university/'` в `vite.config.js`

### Проблема: "Белый экран"
**Решение:** 
1. Проверьте консоль браузера (F12)
2. Убедитесь, что React приложение монтируется
3. Проверьте, что все файлы загружаются
